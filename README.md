# SwiftApply

**SwiftApply** is an innovative tool for generating professional CVs quickly and securely. It leverages AI to optimize resumes according to job descriptions, enhancing your chances of landing your dream job.  

---

## Key Features

- **Automatic CV Generation**: Create a well-structured, professional CV in seconds.
- **Job Description Matching**: Rewrites experience and skills to align with the target job.
- **Pre-designed Templates**
- **Dark/Light Mode**: Fully supported across the project with preference saved locally.
- **Privacy-first Design**: All data is processed locally on the user's device.
- **Suggestions & Feedback**: Users can share ideas or report issues directly through the form.

## Technologies Used

- LLM Integration (GROQ)
- Fast API
- JavaScript (ES6+)
- LocalStorage for user data and theme persistence
- HTML2PDF for exporting CVs
- HTML5 / CSS3 / TailwindCSS

---

## Notes

- User preferences (dark/light mode) and CV data are stored locally in the browser.
- After exporting a CV, the local data is cleared automatically to ensure privacy.
